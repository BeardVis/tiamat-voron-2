# Tiamat CAD Models

This directory contains custom-designed 3D models (Fusion 360 format) used for the *Tiamat* Voron 2.4 build.

## Models Overview

| File | Description |
| --- | --- |
| [`umbilical_mount.f3d`](./umbilical_mount.f3d) | Custom gantry mount for the 4-wire umbilical cable (Power + CAN). Routes the cable from the motherboard to the back-top corner and across the gantry. |
| [`aht30_mount.f3d`](./aht30_mount.f3d) | Bracket for the AHT30 Temperature & Humidity sensor. Designed to be mounted directly onto the `umbilical_mount`. |
| [`sfs_mount.f3d`](./sfs_mount.f3d) | Custom mounting bracket for the BTT Smart Filament Sensor V2.0. |
| [`z_belt_clip.f3d`](./z_belt_clip.f3d) | Secure clip for the Z-axis belt ends, preventing them from dangling after tensioning. |
| [`can_cable_ptfe_clip.f3d`](./can_cable_ptfe_clip.f3d) | Dual-path clip that holds the PTFE filament guide tube together with the CAN bus cable for clean umbilical routing. |
| [`manta_m8p_v2_0_din_clip_back.f3d`](./manta_m8p_v2_0_din_clip_back.f3d) | DIN mount (back) for Manta M8P V2.0. Based on [official PCB DIN clip](https://github.com/VoronDesign/Voron-2/blob/Voron2.4/STLs/Electronics_Bay/pcb_din_clip_x3.stl). |
| [`manta_m8p_din_clip_front.f3d`](./manta_m8p_din_clip_front.f3d) | DIN mount (front) for Manta M8P. Based on [official PCB DIN clip](https://github.com/VoronDesign/Voron-2/blob/Voron2.4/STLs/Electronics_Bay/pcb_din_clip_x3.stl). |
| [`lrs_350_ps_din_clip.f3d`](./lrs_350_ps_din_clip.f3d) | DIN Mount for LRS-350 power supply. Based on [official PCB DIN clip](https://github.com/VoronDesign/Voron-2/blob/Voron2.4/STLs/Electronics_Bay/pcb_din_clip_x3.stl). |
| [`powersupply_side_support.f3d`](./powersupply_side_support.f3d) | Mount for power supply that is mounted on its sides. |
| [`z_belt_cover_panel_support.f3d`](./z_belt_cover_panel_support.f3d) | Modified Z belt cover to support the printer bottom panel. Original: [z_belt_cover_a_x2.stl](https://github.com/VoronDesign/Voron-2/blob/Voron2.4/STLs/Panel_Mounting/z_belt_cover_a_x2.stl). |
